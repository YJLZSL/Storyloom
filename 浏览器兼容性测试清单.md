# 浏览器兼容性测试清单

> AI Timeline Creator V4.0
> 最后更新：2026-06-16
> 用途：为下一任 AI 或测试人员进行全面浏览器兼容性测试提供检查范围

---

## 1. 测试环境要求

| 浏览器 | 最低版本 | 测试重点 |
|--------|----------|----------|
| Chrome | 120+ | 主测试浏览器，验证完整功能 |
| Edge | 120+ | Chromium 内核差异、字体渲染 |
| Firefox | 120+ | CSS 变量、滚动条样式、动画性能 |
| Safari | 17+ (macOS) | backdrop-filter、自定义滚动条、Pointer Events |
| Safari iOS | 17+ | 触摸交互、面板侧滑、虚拟键盘 |

**建议分辨率：**
- 1920×1080（桌面主分辨率）
- 1366×768（笔记本常见分辨率）
- 2560×1440 / 4K（高 DPI 字体与布局）
- 390×844（iPhone 14 Pro 移动端）
