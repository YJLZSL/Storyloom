# Storyloom v1.1.6 更新日志

## v1.1.6 关键修复

### 1. 工作区删除（彻底修复）
- **根因**：服务器端删除工作区时，`crud.ts` 的删除事务中遗漏了 `revisions`、`ai_conversations`、`ai_cache` 等表的清理。虽然这些表部分有 `onDelete: 'cascade'`，但 SQLite 事务中手动删除子表后，遗留的外键约束导致删除失败。
- **修复**：
  - 在删除事务中添加 `revisions`、`ai_conversations`、`ai_cache` 的清理（使用直接 SQL 执行，兼容表不存在的情况）
  - 添加 `try-catch` 错误捕获，返回详细的错误信息给客户端
  - 服务器端添加错误日志记录
- **UI 改善**：Dropdown 中只显示非当前工作区的删除选项（避免误删当前工作区），并显示具体的错误信息 toast

### 2. 隐藏轨道恢复（增强错误处理）
- **修复**：服务器端 `tracks.ts` PATCH 添加 `try-catch` 错误捕获和日志
- **客户端**：恢复按钮添加 `onSuccess`/`onError` toast 提示，显示具体的错误信息

### 3. 左侧搜索事件定位
- **修复**：添加 `leftPanel.searchEventsPlaceholder` 到 `zh-CN.json` 和 `en-US.json`
- **功能**：左侧搜索框改为实时过滤事件，点击后自动切换到时间轴视图并平滑滚动定位

### 4. 时间轴设置标签页
- 新增设置 → 时间轴标签页
- 连接线开关、缩放级别滑块、操作提示（Ctrl+滚轮缩放、滚轮平移、拖拽平移）

## UI/UX 升级

### 5. 字体更换
- **设置 → 偏好 → 界面字体**：支持 5 种字体切换
  - Noto Sans SC（默认）
  - 系统字体
  - Inter
  - 思源黑体
  - 像素风（MC）
- 实现方式：`data-font` 属性 + CSS 变量切换
- 添加 Google Fonts CDN（Inter、Noto Sans SC、Noto Serif SC、Press Start 2P、JetBrains Mono）

### 6. MC 主题番茄钟
- **像素方块风格**：草地绿顶部、泥土棕色边框、方块堆叠效果
- **模式标签**：专注 → "挖掘中"、休息 → "吃面包"、长休 → "长休"
- **进度条**：10 个像素方块组成的进度条，填充颜色随模式变化
- **切换按钮**：MC 主题 ↔ 简约风格一键切换
- **CSS 类**：`.mc-pixel`、`.mc-border`、`.mc-dirt-bg`、`.mc-stone-bg`

### 7. Anime.js 风格 CSS
- 添加 `.anime-gradient`（渐变背景动画）
- 添加 `.anime-glow`（霓虹发光阴影）
- 添加 `.anime-text-glow`（文字发光效果）
