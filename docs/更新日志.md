- **布局修复**：
  - TopToolbar 响应式（overflow-x-auto + 小屏折叠）
  - AppShell flex 约束修复
  - EmptyShell 网格响应式
  - LeftPanel 滚动修复
  - StatusBar 小屏截断修复
  - ContextPanel 使用共享 EmptyState
- **视觉精致化**：
  - 新建 `ConfirmDialog` 组件替代原生 `confirm()`（全局 11 处替换）
  - `WorkspaceCard` 增强（悬停动画、图标丰富化）
  - `EmptyState` 动画增强
  - `ThemeSelector` 预览卡片
- **架构重构**：`OutlineView` 拆分（940 行 → 6 子组件）
  - `useOutlineState.tsx`（538 行）
  - `OutlineHeader.tsx`（85 行）

  - `OutlineTrackSection.tsx`（139 行）

### P0 Bug 修复

- ErrorBoundary 恢复捕获能力
- `api.ts` 竞态条件修复（启动顺序问题）
- `App.tsx` 启动流程修复（从启动遮罩直接进入 workspace）
- `useWorkspaceStore` 死代码 `fetch` 删除
- 后端 `health.ts` 27 表检查增强
- 后端 `crud.ts` 工作区删除修复
- `sidecar-entry.ts` 端口范围扩展（3010-3050）

### 已知限制

- **Sidecar 使用旧版本**：由于当前构建环境 Node v24 不支持 `pkg` 打包工具（pkg 最高支持 Node 18），且无 MSVC BuildTools，无法重新构建 sidecar。因此 v1.1.0 安装包中的 sidecar 仍为旧版本，不包含上述后端修复（health 27 表检查、工作区删除、端口扩展）。这些后端修复将在环境就绪后重新打包。

## v1.0.0 — 统一稳定版（2026-06-22）

### 全新美术架构

- 三层渲染模型：交互层 / 内容层 / 渲染层，PixiJS v8 渐进增强背景渲染
- 动画层架构：GSAP 引擎封装 + 10 种预设 + React Hooks
- 渲染层架构：PixiBackground 组件，8 种粒子预设（织线/星空/萤火/墨滴/棱镜/落樱/气泡/极光）
- 主题系统 v2.0：8 套完整主题配置（颜色/字体/动画/材质），运行时动态切换
- LoomSplash 启动动画：织机 SVG 动画，GSAP Timeline 4 阶段驱动
- PageTransition 页面过渡：10 种过渡预设（淡入淡出/滑动/缩放/旋转/弹性弹出）
- 全面清理 10+ 无用文件和未使用导入
- TypeScript 类型检查：0 错误
- vitest 单元测试：193 / 193 全部通过

### AI 深度集成（v1.3.0）

- AI 对话历史持久化：从 localStorage 迁移到 SQLite 数据库
- AI 工作区上下文注入：自动加载当前工作区的角色、事件、伏笔数据作为 system prompt
- AI 辅助创作增强：事件续写、角色对话、伏笔回收、一致性检查
- 三段式消息结构（系统提示词 + 工作区上下文 + 对话历史）支持 KV 缓存优化
- `aiConversations` / `aiCache` 数据库表已创建并接入后端 CRUD 路由
