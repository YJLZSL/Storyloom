## 1. 工具与 UP 主简介

- **UP 主**：洛笙创作_羽洛笙
- **工具名称**：洛笙创作软件
- **定位**：个人开发的 AI 辅助结构化创作工具，面向小说、剧本、世界观复杂的长篇内容创作。
- **核心卖点**（来自视频标题与简介）：
  - 可视化多线并行时间轴
  - 伏笔追踪与呼应链条
  - AI 辅助创作
  - 立体化整理世界观
  - 细纲演进图
- **获取方式**：关注或评论即可免费下载，核心功能本地免费使用。

## 2. 真实画面特征清单

### 2.1 整体配色与风格

| 特征 | 观察结果 | 截图证据 |
|---|---|---|
| 主背景 | 温暖米白 / 羊皮纸色，接近 `#F7F4ED` / `#F5F0E8` | `bili_timeline_t0420.png` |
| 主按钮 | 绿色（新建、确认）+ 棕色/琥珀色（管理、配置） | `bili_timeline_t0480.png` |
| 强调色 | 轨道/事件条使用蓝、红、橙、紫、绿、灰等高饱和色 | `bili_timeline_t0510.png` |
| 圆角 | 按钮 8–12px，输入框/卡片 6–8px，整体柔和 | `bili_foreshadow_t0060.png` |
| 阴影 | 卡片与按钮轻微弥散阴影，不强烈 | 各对话框截图 |

### 2.2 全局布局

- **顶部一级导航**：横向 Tab 栏，包含「创作中心、角色管理、大纲编排、细纲演进、世界构建、章节目录、资料库、AI助手」等入口。
- **顶部二级工具栏**：当前视图标题 + 搜索框 + 主要操作按钮（绿色）+ 次级按钮（棕色）。
- **左侧边栏**：章节/轨道层级树（演进图中可见卷/章/节列表）。
- **主画布**：
  - 时间轴视图：横向 Gantt 条，轨道头在左，事件条在右；
  - 演进图视图：横向流程图，节点为圆角胶囊。
- **右侧面板**：属性/编辑面板（视频中未完整展示，但工具栏与对话框暗示存在）。

### 2.3 时间轴视图（#3 视频）

- **轨道头**：左侧显示轨道名称，轨道条带颜色背景。
- **事件条**：圆角水平条，长度表示时间跨度，颜色与所属轨道关联。
- **选中态**：当前事件条高亮，其余元素可能降权。
- **工具栏**：「新建事件」「时间线管理」按钮，绿色主按钮 + 棕色次按钮组合。
- **时间体系配置**：弹出对话框可设置时间单位、基础单位、起点，支持跨世纪、自定义历法等宏大世界观。

### 2.4 大纲 / 细纲演进图（#5 视频）

- **大纲编排**：搜索框 + 绿色「新建分组/章节」按钮 + 左侧章节树 + 右侧详情。
- **故事演进图**：
  - 左侧：卷/章/节层级列表；
  - 右侧：横向节点图，节点为圆角胶囊；
  - 绿色节点 = 正常/完成，红色节点 = 冲突/废弃/待处理；
  - 节点间用曲线/折线连接，展示情节依赖与演进。

### 2.5 AI 助手面板（#6 视频片段）

- **大标题卡片**：棕色/主色背景，标题「AI 创作助手」，副标题说明能力范围。
- **功能卡片网格**：4 个等宽卡片，每个含图标、标题、简短说明、操作按钮。
- **功能项**：内容分析、灵感启发、修改建议、结构拆分等。

### 2.6 对话框与表单

- **模态对话框**：居中，白色/米色背景，大圆角（约 12–16px），标题清晰，表单字段垂直排列。
- **表单字段**：标签在上方，输入框/选择器占满宽度，底部右侧放「取消 + 确认」按钮。
- **颜色选择器**：以色块矩阵形式展示，点击选中。
- **配置类对话框**（#1 t=240、#1 t=300、#2 t=240、#6 t=180）：
  - 标题栏左侧有功能图标 + 标题，右侧有关闭按钮；
  - 内容区采用分组卡片或左侧标签页 + 右侧表单的布局；
  - 单选/复选选项使用卡片式宫格，选中态有主色描边或填充；
  - 底部操作区右侧放置「取消 + 确认」，主确认按钮为绿色或棕色。

### 2.7 组件级细节发现（新增）

#### A. 创作中心首页 / 仪表盘（#1 t=0：`bili_basic_t0000.png`）

- **布局**：4 个等宽大卡片组成的网格（2×2），占据主画布中心。
- **单张卡片结构**：
  - 顶部左侧图标 + 标题；
  - 中部统计数字（如「已创建角色 0」）与一句话描述；
  - 底部一个操作按钮。
- **配色**：
  - 角色卡片：棕色按钮；
  - 章节大纲卡片：绿色按钮；
  - 世界要素卡片：棕色按钮；
  - 章节卡片：橙色/琥珀色按钮。
- **可借鉴点**：Storyloom 工作区首页/空状态可复用「图标 + 统计 + 描述 + 彩色按钮」的卡片网格，统一用 `TCard` + `TButton` 实现。

#### B. 细纲总览（#1 t=120：`bili_basic_t0120.png`）

- **三栏布局**：左侧章节树、中间情节点列表、右侧详情/预览。
- **顶部过滤栏**：「全部、起、承、转、合」等阶段标签，绿色「新增节点」主按钮，搜索框。
- **列表项**：每行包含标题、状态色块/标签、操作按钮（编辑/删除）。
- **可借鉴点**：OutlineView 的过滤标签改用 `TTag` 组，列表项操作按钮改用 `TButton` variant="text" + IconPark 图标。

#### C. 世界观资料库（#4 t=120：`bili_world_t0120.png`）

- **顶部统计标签**：「全部 11、待填 2、已填 X、关联中」等状态计数徽章。
- **搜索与过滤**：搜索框 + 下拉筛选器。
- **卡片网格**：每张资料卡片包含：
  - 左上角图标（与分类相关）；
  - 标题与副标题；
  - 状态标签（如「待完善」）；
  - 一段描述文字；
  - 底部标签组或操作按钮。
- **可借鉴点**：WorldBuilding / Character / Connection 等右侧面板可采用资料库卡片网格，用 `TCard` + `TTag` + IconPark 图标统一。

#### D. 世界观思维导图（#4 t=0 / t=60 / t=180：`bili_world_t0000.png`、`bili_world_t0060.png`）

- **中央节点**：深色圆角胶囊，标注「世界观」。
- **分支节点**：不同颜色代表不同维度（地理、历史、势力、种族等）。
- **连线**：柔和的曲线，颜色与分支主题一致。
- **左侧工具栏**：缩放、适应屏幕等按钮。
- **可借鉴点**：RelationshipGraph / ForeshadowingGraph 的节点可借鉴圆角胶囊 + 主题色连线的风格，但保持当前力导向布局。

#### E. AI 助手面板完整结构（#2 t=0：`bili_ai2_t0000.png`；#6 t=0：`bili_aiassist_t0000.png`）

- **顶部横幅**：棕色渐变背景，左侧 AI 图标，标题「AI 创作助手」，副标题「智能分析 · 创意启发 · 写作辅助」；右侧可能有设置/关闭按钮。
- **模型信息条**：横幅下方一行浅色条，显示当前模型、来源、切换模型入口。
- **功能卡片网格**：4 张等宽卡片，每张包含：
  - 顶部大图标；
  - 标题（内容分析 / 灵感启发 / 修改建议 / 结构拆分）；
  - 一行简短说明；
  - 两个底部按钮：左侧主操作（棕色/绿色填充），右侧次级操作（描边/文本）。
- **智能对话区**：卡片网格下方，输入框 + 历史对话列表，输入框右侧有发送按钮。
- **可借鉴点**：当前 `AIPanel.tsx` 可直接复用该布局——`TCard` 网格 + 顶部大横幅 + 底部对话输入框。

#### F. 文件导入对话框（#6 t=60 / t=120：`bili_aiassist_t0060.png`、`bili_aiassist_t0120.png`）

- **拖放区**：虚线边框大矩形，内含上传图标与提示文字，米色背景。
- **文件列表**：导入后显示章节/文件清单，每行有文件名、状态、操作。
- **底部按钮**：「取消」+「开始导入」，主按钮为棕色。
- **可借鉴点**：ImportDialog / ExportDialog 可借鉴该拖放区样式，使用 TDesign Upload 组件并覆盖圆角/边框。

## 4. 本轮改造优先级

### 必须做（P0）
1. ✅ TDesign 主题适配层 + ui-tdesign 扩展。
2. ✅ TopToolbar、SideNav、ContextPanel 核心控件 TDesign 化。
3. ✅ 时间轴工具栏、轨道头、事件卡片 TDesign 化。
4. ✅ IconPark 图标统一，核心布局不再直接使用 lucide-react。
5. ✅ 撰写本文档并归档截图。

### 建议做（P1）
1. ✅ 新增 TimelineMinimap。
2. ✅ OutlineView、命令面板、设置对话框 TDesign 化。
3. ✅ AI 助手面板改为卡片网格布局。
4. ⏳ 其他视图（Narrative/Gantt/Tree/Stats/Relationship）统一按钮/徽章/空状态（主体完成，部分内部图标仍可逐步迁移）。

### 可选做（P2）
1. ⏳ 新增独立「故事演进图」视图（本轮沿用现有关系图/树状图，未新增独立视图）。
2. ⏳ 时间体系配置对话框（自定义历法、跨世纪等）。
3. ✅ 更复杂的节点连线动画（TDesign 化后样式与动效已统一）。
