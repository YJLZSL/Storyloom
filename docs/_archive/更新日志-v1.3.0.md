# Storyloom v1.3.0 更新日志

> 本文档记录 v1.3.0 当前版本的变更。发布日期：2026-06-21

---

## 新功能

- **AI 对话历史持久化**：对话数据从 localStorage 迁移到 SQLite 数据库，支持跨设备同步和大量对话存储
  - 后端新增 `/api/ai/conversations` CRUD 路由（list/get/create/update/delete）
  - 前端 `useAIConversations` hook 重写为 API 驱动，自动加载/保存对话
  - `AIConversationList` 新增搜索过滤和重命名功能
- **AI 工作区上下文注入**：AI 对话自动加载当前工作区的角色、事件、伏笔数据作为 system prompt
  - 三段式消息结构：固定系统提示词 + 工作区上下文 + 对话历史 + 当前消息
  - 支持 KV 缓存优化，大幅降低 API 调用成本
- **AI 辅助创作增强**：新增 4 个一键创作功能
  - **事件续写**：基于选中事件上下文自动扩展描述
  - **角色对话**：生成两个角色之间的自然对话
  - **伏笔回收**：检测未回收伏笔并建议回收方式
  - **一致性检查**：检测时间矛盾、角色行为不一致等逻辑漏洞

## 技术改进

- 新增 `src/lib/ai-context.ts`：AI 上下文构建工具，支持三段式消息结构和 Prompt 模板
- 新增 `src/services/ai-conversations-api.ts`：前端 AI 对话 API 封装
- 后端 `ai_conversations` 表已自动创建（DDL 兜底），老版本升级无感知
- 项目从 Electron 42 迁移至 **Tauri 2.x**，构建产物体积从 ~125MB 降至 ~50MB

## 文档

- 新增 `docs/agents.md`：智能体编码指南，规范 AI 开发流程
- 新增 `plan-v1.3.md`：v1.3 详细开发计划
