## 概述

v1.4.0 是 Storyloom（织叙）的美术架构全面升级版本。在前端全面重构的基础上，引入了**三层渲染模型**（交互层/内容层/渲染层）、**动画层架构**（GSAP 引擎）和**主题系统 v2.0**，将视觉体验从静态主题提升到动态沉浸式境界。

---

## 全新美术架构 v2.0

### 三层渲染模型

将界面视觉分层为三个独立层级：

- **交互层（Interaction Layer）**：用户可操作的 DOM 元素，响应鼠标/键盘事件
- **内容层（Content Layer）**：文本、图像、数据可视化内容，承载信息密度
- **渲染层（Render Layer）**：PixiJS v8 驱动的背景特效层，提供动态视觉氛围

三层之间通过 `RenderBridge` 桥接，实现跨层通信和数据同步。

### 动画层架构

- **AnimationEngine.ts**：GSAP 动画引擎统一封装，支持 Timeline、ScrollTrigger、Stagger 等高级特性
- **动画预设系统**：10 种预设（fade-slide、slide-up、scale-fade、rotate-scale、elastic-pop、slide-left、slide-right、flip-x、flip-y、zoom-rotate）
- **React Hooks**：`useAnimation`（单一元素动画）、`useStaggerAnimation`（交错动画）、`useScrollAnimation`（滚动触发动画）
- **双引擎策略**：Framer Motion 保留用于 DOM 过渡，GSAP 用于复杂时间线动画

### 渲染层架构

- **PixiBackground.tsx**：PixiJS v8 渐进增强背景渲染组件
- **8 种粒子预设**：织线（luosheng）、星空（midnight）、萤火（forest）、墨滴（ink）、棱镜（contrast）、落樱（sakura）、气泡（abyss）、极光（aurora）
- **降级策略**：PixiJS 加载失败时自动降级为 CSS 动画背景
- **性能控制**：根据设备性能和用户偏好动态调整粒子数量

### 主题系统 v2.0

- **8 套完整主题配置**：洛笙（luosheng）、子夜（midnight）、森林（forest）、水墨（ink）、高对比（contrast）、桜（sakura）、深海（abyss）、极光（aurora）
- **每套主题包含**：颜色系统、字体配置、动画参数、渲染层配置、材质纹理
- **动态切换**：运行时切换主题无需刷新，所有层级同步更新

## 核心动画组件

### LoomSplash（启动动画）

- **织机 SVG 动画**：7 条经线 + 4 条纬线交织，模拟真实织机运作
- **GSAP Timeline**：4 阶段动画（经线落下 → 纬线穿梭 → 交织形成 → 品牌浮现）
- **主题自适应**：根据当前主题动态调整颜色
- **跳过机制**：点击或按任意键立即跳过

### PageTransition（页面过渡）

- **10 种过渡预设**：淡入淡出、滑动、缩放、旋转、弹性弹出等
- **GSAP 驱动**：基于 Timeline 的精确时间控制
- **双向动画**：进入/退出动画对称，支持自定义持续时间
