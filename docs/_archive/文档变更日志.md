- **核心布局 TDesign 化**：

## v1.2-backend-implementation（2026-06-19）

**主题**：v1.2 后端路线图全面实施 — 视觉小说数据模型 + CRUD 路由 + 资产存储 + ZIP 导出/导入 + 时光机 + 备份改名 + P0/P1 bug 修复。

### 新增文件
- `server/routes/scenes.ts` — 场景 CRUD（GET/POST/PATCH/DELETE + reorder），含 workspace 存在性校验
- `server/routes/beats.ts` — 节拍 CRUD（scene 作用域），含 reorder
- `server/routes/choices.ts` — 选项 CRUD（beat 作用域，强制 kind=choice 校验）
- `server/routes/flags.ts` — 标志变量 CRUD（name 唯一性校验）
- `server/routes/maps.ts` — 地图 CRUD（含 Konva.js 底图关联）
- `server/routes/assets.ts` — 资产路由（multipart 上传 + SHA-256 寻址去重 + 图片宽高探测 + 缩略图生成 + ETag 流式读取 + 引用计数删除 + 角色/事件/场景资产绑定 3 套 CRUD）
- `server/routes/revisions.ts` — 操作历史/时光机（按 entityType/since 过滤 + 手动创建 + beforeJson 还原端点）
- `drizzle/0003_watery_moon_knight.sql` — VN 9 张表迁移
- `drizzle/0004_spooky_nehzno.sql` — revisions 表迁移
- `docs/接手开发指南 v1.3-remaining-backend-and-frontend.md` — v1.3 交接指南

### 新增依赖
- `@fastify/multipart` — 文件上传（50MB 限制）
- `sharp` — 图片缩略图生成（webp 缓存）
- `archiver` + `@types/archiver` — ZIP 归档（v8 ESM，使用 ZipArchive 类）

### 数据模型变更
- **新增 9 张 VN 表**：assets / maps / scenes / beats / choices / flags / character_assets / event_assets / scene_assets
- **新增 revisions 表**：通用操作历史（entityType + entityId + op + beforeJson + afterJson）
- **数据库现共 21 张表**，迁移文件 0000–0004

### 类型系统变更
- `shared/types.ts`：新增 BeatKind / AssetKind 等联合类型；Scene / Beat / Choice / Flag / Asset / StoryMap / Revision 等接口；全部 Create/Update 请求类型
- `ExportData` 新增 outlineVersions（可选）+ scenes/beats/choices/flags/maps/assets/characterAssets/eventAssets/sceneAssets（可选）

### 校验 Schema 变更
- `server/lib/validation.ts`：新增 sceneIdParam / beatIdParam / choiceIdParam / flagIdParam / assetIdParam / mapIdParam 路由参数
- 新增 createSceneBody / updateSceneBody / createBeatBody 等 body 校验
- 新增 reorderBody（通用排序）/ bindCharacterAssetBody / bindEventAssetBody / bindSceneAssetBody
- 新增 FLEXIBLE_ID 模式（`^.+$`）用于 VN 实体 ID 校验
- **修复**：`foreshadowingIdParam` 从 `foreshadowingId` 改为 `foresId`，`connectionIdParam` 从 `connectionId` 改为 `connId`

### 路由注册
- `server/index.ts`：注册 @fastify/multipart 插件 + 7 个新路由模块
- scenes/flags/maps 注册为 workspace 子资源；beats 注册为 scene 子资源；choices 注册为 beat 子资源
- assets 注册为 workspace 子资源（含 upload/file/thumbnail 端点）
- revisions 注册为 workspace 子资源

### Bug 修复
- **§3.P0-1**：PATCH/DELETE 参数名不匹配（foreshadowingIdParam→foresId, connectionIdParam→connId）
- **§3.P0-2**：workspaces PATCH 漏 settingsJson 字段

- **§3.P1-10**：error-handler.ts JSON 解析错误返回 500（改为 400 + INVALID_JSON code）
- **§2.8**：备份文件前缀 `timeline-creator-` → `storyloom-`（migrate.ts + auto-save.ts，兼容旧文件名过滤）

### 功能增强
- **workspaces export**：支持 `?format=zip`，含所有 VN 表 JSON + 物理资产目录 + README.md
- **workspaces import**：支持 VN 表导入（outlineVersions/assets/maps/scenes/beats/choices/flags/characterAssets/eventAssets/sceneAssets）
