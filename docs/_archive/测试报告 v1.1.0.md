## 4. 自动更新链路验证

应用启动 5 秒后调用 `autoUpdater.checkForUpdates()`，从 GitHub Releases 拉取 `latest.yml`：

```
[updater] checking-for-update
GET https://github.com/YJLZSL/AI-Timeline-Creator/releases/download/v1.0.1/latest.yml
Response: 404 (v1.0.1 release 不含 latest.yml — 这是 v1.0.x 客户端的预期行为)
```

**结论**：
- 自动更新代码路径正常工作，能正确从 GitHub 拉取元数据。
- 本次发布 v1.1.0 时附 `latest.yml`，**所有 v1.1.0+ 客户端都将正常发现后续版本**。
- v1.0.x 客户端没有 electron-updater，需用户主动手动安装本版本。
