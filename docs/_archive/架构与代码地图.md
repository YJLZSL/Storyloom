## 7. 构建与开发

```bash
# 开发（前端 5173 + 后端 3001 同时启动）
npm run dev

# 仅前端
npm run dev:web

# 仅后端（tsx watch 热重载）
npm run dev:server

# Electron 开发模式（加载 localhost:5173）
npm run dev:electron

# 构建
npm run build           # tsc -b && vite build（前端）
npm run build:server    # tsc -p tsconfig.server.json（后端 → dist-server/）
npm run build:electron  # tsc -p tsconfig.electron.json（Electron → electron-out/）

# 完整打包（前端+后端+Electron+electron-builder）
npm run dist

# 类型检查
npm run typecheck       # tsc -b --noEmit

# 数据库
npm run db:generate     # 生成迁移文件（drizzle-kit）
npm run db:migrate      # 执行迁移
npm run db:studio       # 打开 Drizzle Studio
```

**环境分离（骨架搭建后）：**
- 开发环境：`./data/dev.db`，前端 5173 + 后端 3001 + Electron 加载 localhost:5173
- 生产环境：`app.getPath('userData')/timeline.db`，Electron 加载本地 dist/index.html，后端从 dist-server 加载

**Vite 配置要点：**
- 路径别名 `@` → `./src`
- base: `'./'` (Electron file:// 协议兼容)
- 手动分包: react / query / icons 各成独立 chunk
- 开发代理: `/api` → `localhost:3001`
