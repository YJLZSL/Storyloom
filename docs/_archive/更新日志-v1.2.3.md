# Storyloom v1.2.3 更新日志

> 本文档记录 v1.2.3 历史版本的变更。发布日期：2026-06-21

---

## 新功能
- **工作区重命名**：`WorkspaceCard` 新增内联编辑功能，hover 时显示编辑按钮，点击可直接修改工作区名称
- **工作区启动行为设置**：`openLastWorkspace` 设置项已实际生效。关闭后，启动应用时将显示工作区选择器而非直接进入上次工作区
- **创建默认名称**：`CreateWorkspaceDialog` 支持留空名称自动使用默认格式（"新工作区 2026/6/21"）

## 修复
- **工作区选择器冲突**：`TopToolbar` 左上角将原「Dropdown 切换」与「齿轮管理按钮」合并为单一综合下拉菜单，避免功能重叠和用户困惑
- **工作区删除失败**：`WorkspaceSelector` 删除前增加存在性校验，防止删除已不存在的工作区时抛出错误

## UI/UX 优化
- **统一工作区入口**：左上角仅保留一个工作区按钮，下拉菜单集成：工作区列表、新建工作区、管理工作区（回到选择器）
- **管理对话框增强**：`WorkspaceManagerDialog` 新增重命名功能，保留切换/删除/新建能力
