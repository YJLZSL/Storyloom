# 洛笙创作软件 B站视频调研截图

本目录存放对 Bilibili UP 主「洛笙创作_羽洛笙」的《洛笙创作软件》系列教程视频进行调研时截取的关键帧。

## 目录说明

- 截图按视频分组命名，文件名中的 `tXXXX` 表示该截图取自视频的第 `XXXX` 秒。
- 同名 `.json` 文件为对应页面/时刻的浏览器快照或 UP 主空间视频列表数据，非图片。
- 若某张截图的具体画面内容未经确认，则只列出文件名，不做内容描述。
