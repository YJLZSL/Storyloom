# 快捷键大全

Storyloom 支持丰富的键盘快捷键，涵盖视图切换、编辑操作、系统功能等。你可以在「设置 → 快捷键」Tab 中查看和自定义所有快捷键。

---

## 视图切换

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+1` | 切换到时间轴视图 |
| `Ctrl+2` | 切换到大纲视图 |
| `Ctrl+3` | 切换到叙事视图 |
| `Ctrl+4` | 切换到甘特图视图 |
| `Ctrl+5` | 切换到统计视图 |
| `Ctrl+6` | 切换到关系图视图 |

## 编辑操作

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+N` | 新建事件 |
| `Ctrl+S` | 保存 |
| `Ctrl+Z` | 撤销 |
| `Ctrl+Shift+Z` | 重做 |
| `Delete` | 删除选中项 |
| `Enter` | 打开编辑器（选中项） |
| `F2` | 重命名（大纲节点） |
| `Tab` | 降级（大纲节点） |
| `Shift+Tab` | 升级（大纲节点） |

## 时间轴导航

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+=` | 放大时间轴 |
| `Ctrl+-` | 缩小时间轴 |
| `Ctrl+0` | 重置缩放 |
| `←` / `→` | 选中上一个/下一个事件 |
| `↑` / `↓` | 上下移动选中项 |

## 系统功能

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+K` | 打开命令面板 |
| `Ctrl+B` | 切换侧边栏 |
| `F11` | 切换专注模式 |
| `Ctrl+,` | 打开设置 |
| `Ctrl+=` | 放大界面 |
| `Ctrl+-` | 缩小界面 |
| `Ctrl+0` | 重置界面缩放 |

## 自定义快捷键

1. 打开「设置 → 快捷键」Tab
2. 找到要修改的命令
3. 点击当前快捷键，按下新的组合键
4. 如果有冲突，系统会提示冲突的命令

### 支持的修饰键

- `Ctrl` / `Cmd`（macOS）
- `Shift`
- `Alt` / `Option`（macOS）
- `Mod`（自动映射为 Ctrl/Cmd）

### 单键快捷键

部分命令支持单键快捷键（如 `F11`、`Delete`、`Enter`），无需修饰键。

## 平台差异

| 操作 | Windows/Linux | macOS |
|------|---------------|-------|
| 命令面板 | `Ctrl+K` | `Cmd+K` |
| 保存 | `Ctrl+S` | `Cmd+S` |
| 撤销 | `Ctrl+Z` | `Cmd+Z` |
| 重做 | `Ctrl+Shift+Z` | `Cmd+Shift+Z` |
| 新建 | `Ctrl+N` | `Cmd+N` |
| 放大 | `Ctrl+=` | `Cmd+=` |

---

## 最佳实践

1. **优先记忆命令面板**：`Ctrl+K` 是最高效的入口，其他快捷键可以逐步记忆
2. **设置常用命令的快捷键**：为每天使用 10 次以上的命令设置最短的快捷键
3. **避免冲突**：自定义时注意不要与系统快捷键（如 `Ctrl+Alt+Del`）冲突
4. **导出备份**：设置完成后，在「设置 → 关于与数据」中查看快捷键摘要，截图保存
