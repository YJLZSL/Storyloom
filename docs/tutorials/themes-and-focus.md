# 主题与专注模式

Storyloom 提供 6 套精心设计的主题，满足不同创作场景和视觉偏好。

---

## 主题切换

### 切换方式

1. 打开「设置 → 主题」Tab
2. 点击任意主题卡片切换
3. 或使用命令面板搜索主题名称

### 切换动画

切换主题时，Storyloom 使用 View Transitions API 播放圆形展开动画，从点击位置平滑过渡到新主题，带来流畅的视觉体验。

## 主题详解

### 洛圣（默认浅色）

- **背景**：暖米色（#FAF6ED）
- **主色**：暗金色（#B8860B）
- **风格**：古朴、温暖、适合长时间阅读
- **适用**：日间创作、散文、古风小说

### 子夜（深色护眼）

- **背景**：深蓝（#0F172A）
- **主色**：天蓝（#38BDF8）
- **风格**：星空、深邃、低对比护眼
- **适用**：夜间创作、科幻、悬疑

### 森林（舒缓绿调）

- **背景**：淡绿（#E8F0E2）
- **主色**：深绿（#2F7A41）
- **风格**：自然、清新、缓解视觉疲劳
- **适用**：长时间创作、自然主义文学

### 水墨（灰白晕染）

- **背景**：灰白（#F5F5F0）
- **主色**：墨黑（#1A1A1A）
- **风格**：极简、留白、东方美学
- **适用**：追求简洁、剧本创作、 minimalist

### 高对比（纯黑底白字）

- **背景**：纯黑（#000000）
- **主色**：亮黄（#FFFF00）
- **风格**：极端对比、无障碍设计
- **适用**：视力辅助需求、户外强光环境

### 跟随系统

- **自动**：根据操作系统暗色模式设置自动切换「洛圣」或「子夜」
- **智能**：日出/日落时自动切换（如果系统支持）

## 专注模式

专注模式隐藏所有非必要 UI 元素，只留下纯编辑区域，帮助你进入心流状态。

### 开启/关闭

| 方式 | 操作 |
|------|------|
| 快捷键 | `F11` |
| 命令面板 | 搜索「专注模式」 |
| 菜单 | 视图 → 专注模式 |

### 隐藏的元素

- 顶部工具栏
- 左侧导航栏
- 底部状态栏
- 右侧上下文面板

### 保留的元素

- 中央编辑画布
- 极简的退出按钮（鼠标移至屏幕顶部显示）

## 自定义主题（高级）

开发者可通过修改 `src/index.css` 中的主题色块来自定义主题。每个主题定义以下 CSS 变量：

- `--background`：背景色
- `--foreground`：前景文字色
- `--primary`：主色
- `--border`：边框色
- `--muted`：次要背景色
- `--card`：卡片背景色

## 最佳实践

1. **根据时间段切换**：白天用「洛圣」或「森林」，夜间用「子夜」
2. **根据体裁切换**：古风用「洛圣」或「水墨」，科幻用「子夜」，自然文学用「森林」
3. **专注模式 + 子夜**：夜间创作时开启专注模式 + 子夜主题，最小化视觉干扰
4. **高对比辅助**：视力疲劳时临时切换到「高对比」，休息后再切回
