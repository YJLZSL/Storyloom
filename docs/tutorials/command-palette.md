# 命令面板

命令面板是 Storyloom 的「万能搜索框」，让你无需鼠标即可快速执行任何操作。

---

## 打开命令面板

| 快捷键 | 平台 |
|--------|------|
| `Ctrl+K` | Windows / Linux |
| `Cmd+K` | macOS |

---

## 搜索能力

命令面板支持以下搜索类型：

### 1. 切换视图

输入「时间轴」「大纲」「叙事」「甘特」「统计」「关系图」等关键词，快速切换视图。

### 2. 执行命令

输入命令名称或关键词：
- 「新建事件」→ 创建新事件
- 「新建角色」→ 创建新角色
- 「导出」→ 导出工作区
- 「导入」→ 导入工作区
- 「保存」→ 保存当前工作区
- 「撤销」→ 撤销最近操作
- 「重做」→ 重做已撤销的操作

### 3. 切换主题

输入「洛圣」「子夜」「森林」「水墨」「高对比」「跟随系统」切换主题。

### 4. 打开设置

输入「设置」或 `Ctrl+,` 直接打开设置面板。

### 5. 切换专注模式

输入「专注模式」或按 `F11`。

---

## 快捷键提示

每个命令右侧显示对应的快捷键（如果已设置），帮助你记忆常用快捷键。

---

## 使用技巧

1. **模糊搜索**：输入「时」即可匹配「时间轴视图」，无需完整输入
2. **分类浏览**：使用 `↑` / `↓` 在分类间快速导航
3. **直接执行**：输入后按 `Enter` 立即执行第一个匹配结果
4. **关闭面板**：按 `Esc` 或 `Ctrl+K` 再次关闭

---

## 自定义命令

Storyloom 的命令系统支持插件扩展。开发者可通过 `registerCommand()` API 注册自定义命令，命令面板会自动索引新命令。

---

## 与快捷键的联动

命令面板中的命令与快捷键系统共享同一注册表。为常用命令设置快捷键后，命令面板会显示该快捷键，形成「命令面板 ↔ 快捷键」的双向记忆强化。

---

> 💡 提示：养成「想做什么 → Ctrl+K → 输入关键词 → Enter」的习惯，可以大幅减少鼠标操作，提升创作效率。
