# 时间轴视图

时间轴是 Storyloom 的核心视图，按时间顺序排列事件，帮助你把握故事节奏、检查事件冲突、观察多线并行。

---

## 界面布局

时间轴视图由以下部分组成：

- **顶部工具栏**：轨道筛选、缩放控制、日期范围选择、显示选项
- **左侧轨道头**：显示各轨道名称、颜色、可见性开关
- **中央画布**：时间轴主体，事件以卡片形式分布在各轨道上
- **底部迷你图**：全局缩略图，可快速导航到任意时间段
- **右侧上下文面板**（可选）：显示选中事件的详细信息

---

## 轨道系统

轨道（Track）是时间轴的分类维度。每个事件必须属于一个轨道。

### 创建轨道

1. 点击顶部工具栏的「+」按钮
2. 输入轨道名称（如「主线」、「支线A」、「角色成长线」）
3. 选择颜色（8 种预设 + 自定义颜色）
4. 点击创建

### 轨道操作

- **重命名**：双击轨道头名称，或右键选择「重命名」
- **隐藏/显示**：点击轨道头上的眼睛图标
- **修改颜色**：点击调色板图标，选择新颜色
- **删除**：右键选择「删除」，该轨道下的事件将变为「未分类」

### 轨道颜色

8 种预设颜色覆盖常见需求：

| 颜色 | 用途建议 |
|------|----------|
| 赭石 | 主线剧情 |
| 蓝色 | 支线/副线 |
| 绿色 | 角色成长 |
| 红色 | 冲突/危机 |
| 紫色 | 世界观展开 |
| 青色 | 回忆/闪回 |
| 粉色 | 情感线 |
| 橙色 | 伏笔/悬念 |

## 事件卡片

事件是时间轴上的基本单元，显示为矩形卡片。

### 创建事件

1. 在时间轴空白处右键 → 「新建事件」
2. 或使用快捷键 `Ctrl+N`
3. 填写事件信息：标题、摘要、时间、地点、标签

### 事件类型

- **瞬态事件**：仅有开始时间（单点事件）
- **区间事件**：有开始和结束时间（跨度事件）

### 事件操作

- **拖拽**：拖动事件卡片改变时间位置
- **拉伸**：拖动卡片左右边缘改变起止时间
- **跨轨道拖拽**：将事件从 A 轨道拖到 B 轨道
- **双击**：打开事件编辑器
- **右键**：打开上下文菜单（编辑、删除、创建关联、大纲编辑）

| 操作 | 方式 |
|------|------|
| 放大 | `Ctrl+=` 或滚动鼠标滚轮向上 |
| 缩小 | `Ctrl+-` 或滚动鼠标滚轮向下 |

| 水平滚动 | 按住 Shift + 滚轮，或拖动滚动条 |
| 快速跳转 | 点击底部迷你图的任意位置 |

## 连线系统

连线用于表示事件之间的逻辑关系，在画布上以曲线连接两个事件卡片。

### 连线类型

| 类型 | 颜色 | 含义 |
|------|------|------|
| 因果 | 蓝色 | A 导致 B |
| 闪回 | 紫色虚线 | B 是 A 的回忆/闪回 |
| 伏笔 | 绿色虚线 | A 埋设伏笔，B 回收 |
| 平行 | 黄色 | A 与 B 同时发生 |
| 对比 | 红色虚线 | A 与 B 形成对比/对照 |
| 呼应 | 青色 | A 与 B 前后呼应 |
| 转折 | 橙色虚线 | A 是 B 的转折点 |

### 创建连线

1. 在事件 A 上右键 → 「创建关联」
2. 选择关联类型
3. 点击事件 B 完成连接

---

## 显示选项

- **显示连线**：切换是否绘制事件间连线
- **日期范围**：使用日期选择器过滤特定时间段
- **轨道筛选**：仅显示选中的轨道，隐藏其他轨道

## 键盘导航

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+K` | 打开命令面板，搜索事件 |
| `Ctrl+N` | 新建事件 |
| `←` / `→` | 选中上一个/下一个事件 |
| `Enter` | 打开选中事件的编辑器 |
| `Delete` | 删除选中事件 |

## 时间轴最佳实践

1. **用轨道区分叙事线**：主线、支线、角色线各自独立轨道，避免混乱
2. **用颜色编码情感**：危机用红色、温馨用绿色、悬念用橙色，一目了然
3. **区间事件标示持续时间**：旅行、会议、战争等场景用区间事件，而非多个瞬态事件
4. **善用伏笔连线**：埋设伏笔时立刻创建「伏笔」连线指向计划回收的事件，避免遗漏
5. **定期检查冲突**：使用「一致性检查」面板发现时间冲突、逻辑漏洞
