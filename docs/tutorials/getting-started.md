# 入门指南 · Getting Started

> 欢迎使用 Storyloom（絮织）—— 一款为 **视觉小说 / 长篇小说 / 剧本创作者** 设计的本地桌面创作工作台。

Storyloom 将时间轴、大纲、角色关系、伏笔追踪、AI 辅助写作等创作要素整合在同一画布中，帮助你从宏观结构到微观剧本逐层打磨故事。整个应用以 Electron 打包，数据完全本地存储，离线优先，安全可靠。

---

## 1. 安装与启动

### Windows

1. 从 [GitHub Releases](https://github.com/YJLZSL/Storyloom/releases) 下载最新版 `Storyloom Setup X.Y.Z.exe`。
2. 双击运行安装程序，按向导完成安装。
3. 首次启动时，应用会自动初始化本地数据库并检查更新。

### 首次运行

启动后，你会看到 **EmptyShell（空态界面）**：
- 左侧为工作区列表（首次为空）
- 顶栏可切换语言、打开设置、查看关于信息

---

## 2. 工作区：一切故事的起点

在 Storyloom 中，每个项目都是一个**工作区**。你可以在工作区里创建：

- **事件**：时间轴上的节点，带开始/结束时间、地点、标签
- **轨道**：将事件分类（如主线、支线 A、角色线）
- **角色**：人物档案、关系、成长轨迹
- **世界观**：地点、势力、规则设定
- **伏笔**：埋设、发展、回收的全生命周期追踪
- **连线**：事件之间的因果、闪回、伏笔、平行、对比、呼应、转折关系

### 创建工作区

1. 点击「新建工作区」按钮
2. 填写名称和描述（可选）
3. 选择模板（空白、三幕结构、英雄之旅、长篇小说）
4. 点击创建

### 导入 / 导出

- **导出**：选择工作区 → 点击「导出」→ 生成 JSON 文件，包含全部数据
- **导入**：选择工作区 → 点击「导入」→ 选择之前导出的 JSON 文件，支持覆盖/合并/跳过策略
- **自动备份**：开启设置中的「自动保存」后，编辑内容变更后自动保存到本地

## 3. 核心视图一览

工作区创建后，进入主界面。左侧 **SideNav** 可切换以下视图：

| 视图 | 快捷键 | 用途 |
|------|--------|------|
| **时间轴** | `Ctrl+1` | 按时间顺序排列事件，直观呈现故事节奏 |
| **大纲** | `Ctrl+2` | 层级列表组织章节与场景，快速梳理结构 |
| **叙事** | `Ctrl+3` | 聚焦单一场景的台词与演出撰写 |
| **甘特** | `Ctrl+4` | 甘特图形式展示事件跨度与重叠 |
| **统计** | `Ctrl+5` | 数据洞察：角色出场频率、事件分布、伏笔状态 |
| **关系图** | `Ctrl+6` | 角色、事件、世界设定的关联网络 |
| **树状** | — | 将分支选择可视化为一棵树，检查多结局 |

## 4. 命令面板与快捷键

按下 `Ctrl+K`（macOS 为 `Cmd+K`）打开**命令面板**，快速搜索：
- 切换视图
- 新建事件/角色/工作区
- 导出/导入
- 切换主题
- 打开设置

常用快捷键：

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+K` | 打开命令面板 |
| `Ctrl+N` | 新建事件 |
| `Ctrl+S` | 保存 |
| `Ctrl+Z` | 撤销 |
| `Ctrl+Shift+Z` | 重做 |
| `Ctrl+=` / `Ctrl+-` | 放大 / 缩小时间轴 |
| `Ctrl+0` | 重置缩放 |
| `Ctrl+B` | 切换侧边栏 |
| `F11` | 切换专注模式 |
| `Ctrl+,` | 打开设置 |

更多快捷键可在「设置 → 快捷键」Tab 中查看与自定义。

---

## 5. 设置入口

点击顶栏右侧的齿轮图标打开设置面板，包含 7 个 Tab：

- **偏好**：启动时打开上次工作区、自动保存
- **主题**：6 套主题（洛圣/子夜/森林/水墨/高对比/跟随系统）
- **AI**：配置 AI 写作助手（DeepSeek / Kimi / 智谱 / 自定义等）
- **快捷键**：查看与自定义所有快捷键
- **教程**：阅读本系列教程
- **更新**：检查更新、自动更新开关
- **关于与数据**：版本信息、用户数据目录、快捷键摘要
