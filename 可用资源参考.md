## 一、已集成资源

### 1.1 shadcn/ui

- **状态**: ❌ **未集成**（`src/components/ui/` 目录不存在）。早期文档误标为"已集成"，已在 v1.1 审计中纠正。如需使用，需手动执行 `npx shadcn@latest init` 安装。
- **底层**: Radix UI 原语
- **风格**: 与 Tailwind CSS 深度集成，可定制主题变量
- **可用组件**: 尚未安装，需要按需添加 Button, Input, Dialog, Skeleton, Tooltip 等

### 1.2 lucide-react

- **状态**: 已集成
- **图标数量**: ~1500+
- **风格**: 线性描边，统一 24px 网格
- **使用方式**: `import { Save, Trash2, X } from 'lucide-react'`
- **适合**: 工具栏按钮、菜单项、状态图标

### 1.3 anime.js v4

- **状态**: 已集成
- **功能**: animate, stagger, spring, createTimeline, createScope, onScroll
- **体积**: 按需导入约 11-15 KB gzipped
- **配置**: Vite manualChunks 已配置 `vendor-anime` 独立 chunk
- **注意**: v4 API 与 v3 完全不同，不要参考 v3 文档

### 1.4 Zustand 5

- **状态**: 已集成
- **Store 文件**: `src/stores/` 下 appStore, themeStore, panelStore
- **特点**: 轻量、无 boilerplate、支持 selector 优化渲染

### 1.5 @tanstack/react-query 5

- **状态**: 已集成
- **Hooks**: `src/services/api-hooks.ts`
- **特点**: 自动缓存、后台刷新、乐观更新

| npm 包名 | `@icon-park/react` |
| 当前版本 | 1.4.2 |
| 图标总数 | 2658 |

| 许可证 | Apache-2.0 |
| 官方站点 | http://iconpark.bytedance.com/official |

| 分类 | 用途场景 |
|------|----------|

| Peoples | 角色管理（人物、群组、角色关系） |
| Edit | 编辑操作（写入、修改、删除、撤销） |
| Character | 文本/写作相关 |
| Office | 文档、文件夹、笔记 |
| Communicate | 评论、消息、对话 |
| Weather | 场景氛围（晴、雨、雪） |
| Life | 日常生活场景 |
| Emoji | 情绪表达 |
| Base | 通用基础（搜索、设置、收藏） |
| Datas | 数据分析 |
| Charts | 图表可视化 |

**使用方式**:
```tsx

<Clock theme="outline" size="24" fill="#333"/>
<People theme="filled" size="20" fill="currentColor"/>
```

## 二.5、国产 AI API 参考

### DeepSeek
- **官网**: https://platform.deepseek.com/
- **API Base URL**: `https://api.deepseek.com`
- **兼容协议**: OpenAI 兼容
- **主要模型**:
  - `deepseek-chat` — 通用对话（V3）
  - `deepseek-reasoner` — 推理模型（R1）
- **免费额度**: 有免费额度，具体以官网为准
- **API 文档**: https://platform.deepseek.com/api-docs/

### Kimi (Moonshot)
- **官网**: https://platform.moonshot.cn/
- **API Base URL**: `https://api.moonshot.cn/v1`
- **兼容协议**: OpenAI 兼容
- **主要模型**:
  - `moonshot-v1-8k` — 8K 上下文
  - `moonshot-v1-32k` — 32K 上下文
  - `moonshot-v1-128k` — 128K 上下文
- **特色**: 超长上下文支持，适合长篇小说分析
- **API 文档**: https://platform.moonshot.cn/docs/

### MiniMax
- **官网**: https://platform.minimaxi.com/
- **API Base URL**: `https://api.minimax.chat/v1`
- **兼容协议**: OpenAI 兼容
- **主要模型**:
  - `MiniMax-Text-01` — 通用文本模型
  - `abab6.5s-chat` — 对话模型
- **特色**: 语音合成能力强
- **API 文档**: https://platform.minimaxi.com/document/

### 智谱 GLM
- **官网**: https://open.bigmodel.cn/
- **API Base URL**: `https://open.bigmodel.cn/api/paas/v4/`
- **兼容协议**: OpenAI 兼容
- **主要模型**:
  - `glm-4-flash` — 免费快速模型
  - `glm-4-plus` — 增强模型
  - `glm-4-long` — 长上下文模型
  - `glm-4-air` — 轻量模型
- **免费额度**: glm-4-flash 免费
- **API 文档**: https://open.bigmodel.cn/dev/api/

> **注意**: 以上所有国产 AI 提供商均兼容 OpenAI API 协议，可直接使用 `openai` SDK 或兼容客户端调用，只需修改 `base_url` 和 `api_key`。
