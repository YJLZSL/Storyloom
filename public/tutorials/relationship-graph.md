# 关系图教程

## 功能概述

关系图是 Storyloom（织叙）的核心可视化模块之一，它基于 D3.js 力导向图技术，将角色、势力、世界观设定等叙事元素以网络节点的形式呈现，并通过连线展示它们之间的复杂关联。无论是人物间的情感纠葛、势力间的明争暗斗，还是世界观设定之间的因果呼应，关系图都能帮助创作者一眼洞察全局，发现叙事中的断裂点或冗余连接，从而优化故事结构。

在关系图中，每个节点代表一个叙事实体（如角色、组织、地点、事件等），而连线则代表实体之间的关系类型。Storyloom 内置了多种专业叙事关系标签，包括因果、闪回、伏笔、平行、对比、呼应、转折等，让创作者能够以编剧和小说家的专业视角管理故事网络。

## 操作指南

### 查看关系网络

1. 进入"关系图"面板后，系统会自动加载当前项目中的所有已注册实体并生成力导向布局。
2. 鼠标悬停在任意节点上，即可高亮显示该节点的所有直接关联连线及相邻节点，其他无关元素会自动淡化，方便聚焦观察。
3. 按住鼠标左键拖动节点，可以手动调整布局；力导向模拟会自动重新平衡相邻节点位置。
4. 使用鼠标滚轮可缩放画布，按住画布空白处拖动可平移视角，以便浏览大型关系网络的不同区域。
5. 点击节点可打开右侧详情面板，查看该实体的基础信息、关联统计以及详细属性。

### 添加连接

1. 在画布上方工具栏中点击"添加连线"按钮，或选中一个节点后右键选择"从此节点新建连接"。
2. 将鼠标移动到目标节点上并点击，系统会弹出关系类型选择框。
3. 从下拉菜单中选择关系类型：
   - **因果**：表示事件 A 直接导致了事件 B 的发生，适用于情节推进线的梳理。
   - **闪回**：表示当前节点与过去某一事件存在回忆性关联，常用于非线性叙事结构。
   - **伏笔**：表示节点 A 为节点 B 的后续发展埋下了线索，帮助检查前期铺垫是否充分。
   - **平行**：表示两条剧情线同时发生，彼此独立但可形成蒙太奇效果。
   - **对比**：表示两个节点之间存在鲜明反差，用于强化主题张力。
   - **呼应**：表示前后节点形成首尾照应，增强叙事整体性。
   - **转折**：表示前一状态因该连接而发生剧变，标记关键情节点。
4. 选择关系类型后，可填写额外的连接说明（如具体章节、场景描述），然后点击确认保存。
5. 新连线会实时出现在画布中，并带有对应的颜色标识，不同类型关系拥有不同颜色，便于快速区分。

### 筛选节点

1. 点击画布左上角的"筛选"图标，打开节点筛选面板。
2. 可按实体类型筛选：勾选或取消勾选"角色""势力""地点""事件""道具""概念"等类别，实时显示或隐藏对应节点。
3. 可按关系类型筛选：例如只显示"因果"和"伏笔"关系，隐藏其他连线，以专注于情节逻辑链的梳理。
4. 支持按关键词搜索节点名称，搜索结果会自动定位并高亮对应节点。
5. 可保存当前筛选配置为快照，方便在不同创作阶段快速切换视角（如"只看主线因果""只看角色情感"等）。
6. 点击"重置"按钮可一键恢复全量显示。

### 编辑与删除

1. 选中已有连线，右侧面板会显示该连线的关系类型及说明。
2. 点击"编辑"可修改关系类型或更新说明文字；点击"删除"可移除该连接，系统会提示确认以避免误操作。
3. 批量操作：按住 `Ctrl` 键可多选节点或连线，然后统一修改颜色、类型或删除。

## 快捷键

| 快捷键 | 功能说明 |
|--------|----------|
| `Ctrl + N` | 新建节点（在当前画布中心位置） |
| `Ctrl + Shift + L` | 新建连线（进入连线模式） |
| `Delete` / `Backspace` | 删除选中的节点或连线 |
| `Ctrl + F` | 打开节点搜索框 |
| `空格键` | 暂停/恢复力导向模拟动画（方便固定布局截图） |
| `Ctrl + 0` | 重置画布缩放比例，居中显示全部节点 |
| `Ctrl + S` | 保存当前关系图布局与筛选状态 |
| `滚轮上/下` | 缩放画布 |
| `按住中键拖动` | 平移画布 |

## 最佳实践

1. **分阶段梳理**：在项目初期，先建立主要角色和核心势力的节点，不必一次填满所有细节；随着剧情推进，逐步添加次要角色和事件节点，保持关系图的清晰可读性。

2. **善用颜色与关系类型**：为不同关系类型赋予一致的语义逻辑，例如所有"伏笔"连线使用虚线样式，"因果"连线使用实线箭头。在复杂项目中，这能大幅降低认知负担。

3. **定期审查断裂节点**：每月或每完成一个大章节后，检查关系图中是否存在没有任何连接的孤立节点。孤立节点通常意味着该角色或设定尚未充分融入叙事，需要补充关联或考虑删除。

4. **利用筛选进行专题分析**：在检查伏笔回收情况时，只显示"伏笔"和"呼应"关系；在梳理多线叙事时，只显示"平行"关系。通过筛选器将复杂网络拆解为多个子图，逐一验证其逻辑完整性。

5. **导出与分享**：关系图支持导出为 PNG/SVG 图片格式，建议在关键里程碑节点（如大纲定稿、第一稿完成）导出存档，作为团队沟通或投稿提案的辅助材料。

6. **结合剧本编辑器使用**：在关系图中发现情节断裂后，可直接双击对应事件节点，跳转至剧本编辑器中该场景的位置进行修补，实现从宏观结构到微观文本的无缝衔接。
