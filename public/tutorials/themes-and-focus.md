# 主题与专注模式

Storyloom（织叙）提供丰富的视觉主题与专注写作工具，帮助你在不同场景下找到最舒适的创作状态——无论是深夜的暗色写作、全屏的沉浸式创作，还是需要节奏管理的番茄钟。

## 功能概述

### 主题系统

Storyloom 内置多种精心调校的主题，涵盖不同使用场景：

- **默认明亮**：干净的白底黑字，适合白天和良好光线环境
- **夜间模式**：深色背景搭配柔和的暖色文字，降低夜间写作的视觉疲劳
- **护眼暖黄**：纸黄色的编辑器背景，模拟纸质书的阅读体验
- **高对比度**：强对比度设计，适合视力敏感或户外使用场景
- **自定义主题**：支持调整编辑器背景色、文字色、强调色、侧边栏配色等

所有主题都经过**叙事写作场景**优化，确保长时间盯着文字时眼睛不易疲劳，同时保持代码/标记的清晰可读。

### 禅模式（Zen Mode）

禅模式是 Storyloom 的**全屏沉浸式写作模式**，它会：

- 隐藏所有侧边栏、工具栏和导航元素
- 将编辑器扩展到全屏，最大化可视区域
- 淡化所有非编辑区域，让注意力完全集中在文字上
- 保留最小化的必要控件（如字数统计、退出按钮）
- 支持行聚焦（高亮当前行，淡化其他行）

### 番茄钟

内置的番茄钟帮助你以**时间块**的方式管理写作节奏：

- 支持自定义专注时长（默认 25 分钟）和休息时长（默认 5 分钟）
- 提供完成音效和通知提醒
- 自动记录每日完成的番茄数
- 支持「长休息」设置（每 4 个番茄后自动进入 15 分钟长休息）

## 操作指南

### 切换主题

**方式一：通过命令面板**

1. 按 `Ctrl + K` 打开命令面板
2. 输入「主题」或「theme」
3. 选择「切换主题」
4. 在弹出的主题选择器中点击想要的主题

**方式二：通过设置菜单**

1. 点击右上角「设置」图标（齿轮）
2. 在左侧选择「外观」
3. 在「主题」下拉框中选择目标主题
4. 切换会即时生效，无需重启

**方式三：自定义主题**

1. 进入「设置」→「外观」
2. 选择「自定义」主题
3. 点击「编辑配色」展开调色面板
4. 分别调整：
   - **编辑器背景**：主写作区域底色
   - **文字颜色**：正文文字颜色
   - **强调色**：链接、按钮、高亮等交互元素的颜色
   - **侧边栏背景**：左侧导航栏底色
   - **边框颜色**：分隔线、卡片边框等
5. 调整完成后，点击「保存为自定义主题」

> **提示**：自定义主题可以导出为 `.json` 文件分享，也可以从文件导入他人分享的主题。

### 进入/退出禅模式

**进入禅模式**：

1. 按 `Ctrl + Shift + Z` 快捷键（或在命令面板搜索「禅模式」）
2. 整个界面会平滑过渡到全屏写作状态，所有干扰元素淡出

在禅模式中：
- 编辑器占据屏幕中央，左右留出舒适的边距
- 顶部显示当前章节名称和实时字数统计
- 右下角有一个「退出」按钮
- 可以使用 **行聚焦** 功能：按 `Alt + F` 开启/关闭，此时只有光标所在行保持正常亮度，其余行会变淡

**退出禅模式**：

1. 按 `Esc` 键
2. 或点击右下角的「退出禅模式」按钮
3. 或按 `Ctrl + Shift + Z` 再次切换

> **禅模式技巧**：在禅模式下仍然可以使用命令面板（`Ctrl + K`）和 AI 面板（`Ctrl + Shift + A`），无需退出即可调用辅助功能。

### 使用番茄钟

**启动番茄钟**：

1. 点击顶部工具栏的「番茄钟」图标（番茄形状），或在命令面板搜索「番茄钟」
2. 弹出的计时器面板会显示：
   - 当前倒计时
   - 今日已完成番茄数
   - 控制按钮（开始/暂停/重置）
3. 点击「开始」启动计时

**设置番茄钟**：

1. 在计时器面板中点击「设置」齿轮图标
2. 调整参数：
   - **专注时长**：建议 25-45 分钟
   - **短休息时长**：建议 3-5 分钟
   - **长休息时长**：建议 15-30 分钟
   - **长休息间隔**：几个番茄后进入长休息（默认 4）
   - **音效**：开启/关闭计时结束提示音
3. 点击「保存」

**番茄钟使用流程**：

1. 开始写作前启动一个番茄钟
2. 在专注期间，全身心投入写作，不做其他事情
3. 计时结束时，Storyloom 会播放提示音并弹出通知
4. 利用休息时间站起来活动、喝水、眺望远方
5. 休息结束后，开始下一个番茄钟

> **提示**：开启「禅模式 + 番茄钟」的组合可以获得极致的专注体验。番茄钟会隐藏在禅模式的角落，以极简方式显示剩余时间。

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + Shift + Z` | 进入/退出禅模式 |
| `Alt + F` | 在禅模式中开启/关闭行聚焦 |
| `Ctrl + Shift + T` | 打开/关闭番茄钟面板 |
| `Ctrl + K` → 输入「主题」 | 切换主题 |
| `Esc` | 在禅模式中退出禅模式 |

## 最佳实践

1. **根据时间选择主题**：白天使用明亮主题，傍晚切换为护眼暖黄，深夜使用夜间模式。自动切换功能可以在「设置」→「外观」中配置，根据系统时间自动切换主题。

2. **禅模式用于「冲刺写作」**：当你需要完成一段高强度创作（如一个场景的初稿）时，进入禅模式并配合番茄钟。退出禅模式后，再进行编辑、整理和润色工作。

3. **番茄钟不是越多越好**：建议每天设定合理的番茄数目标（如 8-12 个番茄），而非无休止地追求数量。质量比数量更重要。

4. **自定义你的理想配色**：如果你在夜间写作时仍感到眼疲劳，尝试自定义主题。降低对比度（不使用纯白配纯黑）、增加文字行距、选择柔和的强调色都能提升舒适度。

5. **长休息时不要盯着屏幕**：当番茄钟提示长休息时，真正离开电脑。这是保护颈椎和眼睛的最佳时机，也是让大脑潜意识继续处理故事创意的间隙。
