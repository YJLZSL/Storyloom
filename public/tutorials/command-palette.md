# 命令面板

命令面板是 Storyloom（织叙）的「中央控制台」，通过统一的搜索入口，你可以快速执行任何操作、跳转到任何位置，无需在菜单中层层翻找。

## 功能概述

命令面板将 Storyloom 中所有功能整合为一个可搜索的列表，核心特性包括：

- **全局搜索**：通过关键词模糊匹配命令名称、功能描述和所属分类
- **快速跳转**：直接跳转到特定章节、角色、世界观页面等
- **快捷操作**：一键执行新建、导出、切换视图等操作
- **实时过滤**：输入即过滤，支持拼音和首字母匹配
- **命令历史**：自动记录最近使用的命令，优先展示

命令面板的设计理念是「**不离开键盘**」——任何鼠标操作都可以通过一次快捷键 + 几个字母完成。

## 操作指南

### 打开命令面板

有两种方式可以打开命令面板：

1. **快捷键**：按下 `Ctrl + K`（macOS 为 `Cmd + K`）
2. **菜单入口**：点击顶部导航栏中的「搜索」图标

命令面板会以浮层形式居中显示，当前页面内容会轻微模糊，帮助你聚焦在搜索任务上。

### 搜索命令

打开命令面板后，输入框会自动获得焦点。直接输入你想查找的内容的关键词：

- 输入「主题」→ 显示「切换主题」「导入主题」「重置主题」等相关命令
- 输入「导出」→ 显示「导出为 Markdown」「导出为 WebGAL」「导出为 Word」等
- 输入「角色」→ 显示「新建角色」「打开角色列表」「导入角色」等
- 输入「zms」→ 通过拼音首字母匹配到「**主题设置**」

**搜索结果的组成**：

| 元素 | 说明 |
|------|------|
| 图标 | 每个命令左侧都有类别图标（如 📝 文档、⚙️ 设置、🔄 动作） |
| 命令名称 | 高亮匹配到的关键词 |
| 快捷键 | 如果该命令有快捷键，会在右侧显示 |
| 分类标签 | 标明该命令属于哪个模块（如「编辑」「视图」「工具」） |

使用 **↑ / ↓** 方向键在结果中移动，按 **Enter** 执行选中的命令。按 **Esc** 关闭命令面板。

### 快速导航

命令面板不仅是「执行命令」的工具，更是「**快速跳转**」的利器：

1. 打开命令面板
2. 输入你想跳转到的内容名称，例如：「第三章 命运的分岔」
3. 搜索结果会显示：
   - 匹配的**章节**（点击直接打开该章节编辑器）
   - 匹配的**角色**（点击打开角色卡片）
   - 匹配的**世界观文档**（点击打开对应文档）
   - 匹配的**标签**（点击打开标签筛选视图）

这意味着你无需在侧边栏的文件夹中翻找，只要记得名字（或部分名字），就能瞬间抵达。

### 命令分类浏览

如果不确定关键词，也可以按分类浏览：

1. 打开命令面板后，清空输入框（或按 `Ctrl + K` 再次打开）
2. 输入 `>` 符号，会显示所有可用的命令分类：
   - `> 编辑` — 与内容编辑相关的命令
   - `> 视图` — 与界面显示相关的命令
   - `> 工具` — 与 AI、番茄钟、备份等工具相关的命令
   - `> 文件` — 与导入、导出、新建等文件操作相关的命令
   - `> 偏好` — 与设置、主题、快捷键相关的命令
3. 选择分类后，会列出该分类下的所有命令

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + K` | 打开命令面板 |
| `Esc` | 关闭命令面板 |
| `↑` / `↓` | 在搜索结果中上下移动 |
| `Enter` | 执行选中的命令/跳转 |
| `Ctrl + Shift + K` | 打开命令面板并自动聚焦到「最近使用」 |

> **注意**：部分命令在特定上下文下会有额外快捷键。例如当编辑器获得焦点时，`Ctrl + S` 会保存当前文档；在命令面板中搜索「保存」也可以执行相同操作。

## 最佳实践

1. **记住「Ctrl + K」这一个快捷键**：Storyloom 的大部分功能都可以通过命令面板触达。与其记忆十几个快捷键，不如只记住 `Ctrl + K`，然后输入功能名称。

2. **使用模糊匹配**：不需要输入完整命令名。例如想打开「专注模式（禅模式）」，输入「zms」「禅」「专注」甚至「zen」都能匹配到。

3. **善用最近使用**：命令面板会智能记录你最近使用过的命令，并将它们排在搜索结果前面。如果你反复使用某个命令（如「导出 WebGAL」），几次之后它会在输入框为空时就出现在列表顶部。

4. **快速切换章节**：长篇创作中频繁在不同章节间切换时，用 `Ctrl + K` 输入章节名，比用鼠标在侧边栏翻找快得多。

5. **结合导航与操作**：在命令面板中，输入 `>` 浏览分类命令，输入 `@` 快速搜索角色，输入 `#` 快速搜索标签，输入 `!` 快速搜索世界观文档。这些前缀符号可以帮你精准定位到想要的内容类型。
