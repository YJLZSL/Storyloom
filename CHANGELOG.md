## [1.0.0] — 2026-06-22

**Storyloom · 织叙 v1.0.0 — 统一稳定版**

> 品牌：**Storyloom · 织叙** — 用一根时间线，把故事织成宇宙。
> 面向小说作者、编剧与游戏叙事设计师的本地桌面创作工作台。

### 创作工具

- **时间轴视图**：7 种模式（时间轴/大纲/叙事/甘特/树状/统计/关系图），支持缩放、拖拽、轨道管理
- **角色系统**：角色面板、角色-事件关联、角色特征管理
- **伏笔系统**：伏笔看板、伏笔回收图、未回收检测
- **世界观设定**：WorldBuilding 面板、世界观元素关联
- **关联线**：事件间关联、连接线可视化、关联类型
- **书签系统**：为时间轴事件添加书签，自定义颜色/名称，一键跳转
- **地图系统**：创建地图标记故事地点，标记点关联事件

### 写作与编辑

- **Zen Mode**：全屏沉浸式编辑，悬浮工具栏，实时字数统计
- **三栏写作布局**：左章节目录 + 中编辑器 + 右数据面板
- **每日目标字数**：默认 2000 字/天，自动跨天重置，连续达标统计
- **番茄钟**：Minecraft 像素风格 + 简约风格可切换
- **VN/剧本编辑器**：场景/节拍/选项编辑（后端 API 已就绪，前端待补全）

### 资料库

- **笔记管理**：NoteEditor + FolderTree + 标签系统
- **快速记录**：Ctrl+Enter 快速新建笔记
- **文件夹树**：层级化笔记组织

### AI 助手

- **7 个 AI 提供商**：OpenAI / Anthropic / Google / DeepSeek / 通义千问 / 智谱 / 月之暗面
- **对话历史持久化**：SQLite 存储，跨会话保留
- **工作区上下文注入**：自动加载当前工作区角色/事件/伏笔作为 system prompt
- **场景化功能卡片**：内容分析、灵感启发、修改建议、结构化拆分
- **辅助创作**：事件续写、角色对话、伏笔回收、一致性检查

### 视觉体验

- **三层渲染模型**：交互层 / 内容层 / 渲染层，PixiJS v8 渐进增强
- **8 套主题**：洛笙/子夜/森林/水墨/高对比/桜/深海/极光，运行时动态切换
- **动画系统**：GSAP 引擎 + 10 种预设 + React Hooks
- **PixiJS 粒子背景**：8 种粒子预设（织线/星空/萤火/墨滴/棱镜/落樱/气泡/极光）
- **LoomSplash 启动动画**：织机 SVG 四阶段动画
- **PageTransition**：10 种页面过渡预设
- **Glassmorphism 2.0**：毛玻璃、弹性动画、发光选中态

### 基础设施

- **Tauri 2.x 桌面壳**：~27 MB 安装包，系统 WebView2
- **Sidecar 架构**：Node.js Fastify 后端独立进程，stdout JSON 协议通信
- **自动更新**：minisign 签名 + tauri-plugin-updater
- **NSIS 定制安装程序**：品牌欢迎页、中文语言包、桌面/开始菜单快捷方式
- **命令面板**：Spotlight 风格全局搜索 + 快捷键
- **国际化**：中文/英文双语（react-i18next）
- **数据库**：SQLite 28 表，DDL 兜底迁移，健康检查端点

### 布局与交互

- **顶部水平导航**：品牌 Logo + 导航 Tab + 操作按钮（`TopToolbar`）
- **统一布局模式**：`PageHeader` → `SearchBar` → `StatsBar` → 内容
- **左侧边栏**：仅在时间轴视图显示，可折叠
- **关联数据面板**：右侧面板自动展示选中事件的关联角色/伏笔/事件
- **响应式**：移动端抽屉/Sheet 布局（<768px 适配）
- **骨架屏**：Timeline/Workspace/Stats 数据加载骨架屏
- **空状态**：SVG 编织动画 + 引导文字

### 构建质量

- TypeScript 编译：0 错误
- 单元测试：193 / 193 通过（vitest，22 个测试文件）
- 安装程序：~27 MB（Tauri NSIS）
